# Image to Text OCR Pipeline

## 🚀 プロジェクト概要
Tesseract OCR を使用して画像からテキストを抽出し、クリーンアップする Python スクリプト

- **画像前処理（グレースケール、コントラスト強調）を適用**
- **OCR後のノイズ除去（正規表現によるフィルタリング）を実装**
- **OCRの生データとクリーン済みデータを両方保存**
- **GitHub へのプッシュ準備完了**
